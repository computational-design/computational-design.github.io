Hardware

The Hardware unit is meant to teach the basic functions of the components in a computer. In this unit, our class disassembled various devices to learn what hardware was needed for each device. As we took out the components, we discussed what we thought each piece was, and then looked up the actual answer to see how close we were. By doing this, we learned about the progression of each component as technology evolves, and the companies that produce them. Our final project for this unit was a videogame which included the definitions and pictures of common components in laptops, iPhones, and Desktop PCs. You can find it linked below the images.
